# 平衡性报告（方法论重置后 · 持续更新）

> ⚠️ **本轮数据与前两轮报告不可直接对比**，因为测量工具本身换了四样东西：
> 1. **贪心 AI 重写**：手写期望公式 → 1-ply 真实引擎推演 + 局面评估（`ai.ts` + `evaluate.ts`）。
>    旧 AI 对多个签名技能使用率为 0%（辅助/防御型签名按 category 兜底打分被弃用），
>    旧报告里这些角色的胜率实际只测了"白板属性+被动+普攻"。
> 2. **引擎 bug 修复**：`acBonus` 原先只加不减（"本回合 AC+3"实为永久叠加），已加 `acBonusTurns` 衰减。
> 3. **3v3 指标更换**：固定基准队替换位 → **随机组队选秀价值**（蒙特卡洛，见下）。
>    旧指标分数被"被换下的人有多强"主导，基准队成员自带 ~83% 主场偏差。
> 4. **对局上限**：runMatch guard 5000→800（超长局实质为双回血死局，计平局）。

## 方法论

- 数据：`npm run sim:balance 10 200`。每角色**只带签名 + 被动**（隔离 kit 强度）。
- **1v1 对轰**：12×12 循环赛，每组合 200 场（每行 2400 场，噪声 ≈±2pt）。
- **3v3 选秀价值**：每角色放进 40 支**随机三人队**，对阵全员共用的随机敌队组，
  每对阵 8 场取平均胜率（每角色 320 场，噪声 ≈±5pt）。联动（CA+BC 同队）自然进样本。
- **技能使用率**：AI 健康度仪表盘。某角色签名使用率 ≈0 ⇒ 其胜率数据不反映 kit 强度
  （要先查 AI/技能定价，再谈平衡）。本轮全员 ≥2%，数据有效。

## CA↔BC 联动专项（`sim.cli.ts pair`）

强制 CA+BC 同队（第三人随机，40 队 × 8 场，敌队与选秀价值同种子可比）：

|---|---:|---:|---:|
| 重做前（×1.1/×1.3，华尔兹无伤害） | 43% / 41% | 39% | **−4pt（反协同）** |

旧机制反协同的原因（已修复）：×1.1 增益抵不过占位成本；×1.3 死亡条款奖励舞伴去死；
治疗联动在纯签名 build 下字面不可达（无 heal 来源）。

重做副作用：华尔兹伤害增益让 **BC 自身全队通用**地变强（3v3 41→57%，升至第 3），
不只服务 CA——属可接受的连带 buff，但 BC 进入观察名单。
CA 单独仍 30/41（被动只在 BC 同队时激活，属"成对角色"的设计代价）。

## 技能消融实验（`sim.cli.ts ablate`，Lv12，40 队 × 8 场/变体，配对比较）

**通用技能 add-one**（全员附学该技能 vs 纯签名对照 57%；Δ=胜率贡献，噪声 ≈±3pt）：

| 技能 | Δ | 技能 | Δ |
|---|---:|---|---:|
| brave_strike 英勇打击 | **+34pt** 🔴 | war_cry 战吼 | +3pt |
| charge_smash 蓄力重击 | **+24pt** 🔴 | feint 佯攻 | +2pt |
| heal 治疗术 | +17pt | revive 复活术 | +2pt |
| flurry 疾风连击 | +13pt | firestorm 烈焰风暴 | +1pt |
| precise_aim 精准瞄准 | +10pt | stun_strike 眩晕突袭 | **−1pt** 🔴 |
| shield_block 护盾格挡 | +7pt | | |

> 口径注意：「全员同学」放大可叠加的团队技能（heal×3）；rally 类不叠加（war_cry 3 份浪费 2 份）
> 故 war_cry 略被低估。revive 在「无人倒地就是死技能」的局里天然低频。

**结论**：技能池严重分层。brave_strike 是 +7 伤害/能量的怪物（cost 1 拿 3d6+命中2），
charge_smash 次之；而 stun_strike / firestorm / feint / revive 接近零贡献——**半个技能池是陷阱选项**。

**三轮 + 3b 补丁后复测**（同口径）：
brave_strike +34→**+11~19**（回落到 heal/flurry 梯队）✅；
charge_smash +24→+0（去必中矫枉过正）→ 3b 降费 cost 3→2 后 **+6** ✅（弃回合技能的健康位）；
revive +2→**+28** ⚠ 偏高（注意口径放大：全员 3 份复活 ≈ 不死之队，单份实际更低；候选 15%→10%）；
firestorm +1→**+21**（灼烧给足了存在感，略高可观察）；
Trippi 签名 Δ≈0→**+3pt**、1v1 41→57%（命中即眩晕在单挑里是控制链，处于正常区间上沿）；
斩首解耦后 CA 签名 **+18pt**、地毯降费后 Bombardiro 签名 **+15pt**。

**签名技能 remove-one**（仅对拥有者；带 vs 剥离签名，被动保留）：

| 角色 | with/without | Δ | 角色 | with/without | Δ |
|---|---|---:|---|---|---:|
| Tung | 60/38 | **+22pt** | Boneca | 51/36 | +15pt |
| Bombombini | 46/24 | **+22pt** | Tralalero | 42/27 | +14pt |
| Ballerina | 53/34 | +19pt | Frigo | 31/20 | +11pt |
| CA | 58/43 | +16pt | Bombardiro | 46/37 | +10pt |
| Chimpanzini | 25/10 | +16pt | Patapim | 55/48 | +8pt |
| | | | Lirilì | 56/53 | +3pt |
| | | | **Trippi** | 73/74 | **−2pt** 🔴 |

**结论**：签名设计基本健康（11/12 正贡献，Δ 中位数 ~+15pt）。两个异常：
**Trippi 哈气 Δ≈0**——它的强度全在九命被动（without 仍 74%），哈气两轮 buff 后依旧鸡肋，
建议换设计而非继续加数值；**Lirilì 时间静止 +3pt** 偏低，可观察。
Chimpanzini 反向值得注意：剥离签名只剩 10%，整个 kit 系于狂猿一身。
